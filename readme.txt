Le notebook "PoC_0_shot_classification.ipynb" démontre une implémentation de la classification sans exemple (zero-shot classification) utilisant un modèle pré-entraîné. Le code inclut les étapes principales suivantes :

Chargement des données : Préparation et chargement des données textuelles à classifier.
Application du modèle de classification sans exemple : Utilisation d'un modèle pour classifier les textes en différentes catégories sans nécessiter d'étiquettes préalablement définies.
Analyse des résultats : Évaluation et visualisation des résultats de la classification pour déterminer l'efficacité du modèle.
